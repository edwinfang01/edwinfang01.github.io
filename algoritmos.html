Principales Algoritmos para Manejo de Cadenas

A continuación, se presentan algunos algoritmos comunes utilizados en distintos lenguajes de programación para trabajar con cadenas.

1. Reversión de una Cadena

Este algoritmo invierte el contenido de una cadena.

JavaScript


function reversaCadena(str) {
    return str.split("").reverse().join("");
}
console.log(reversaCadena("Hola"));

Python


def reversa_cadena(texto):
    return texto[::-1]

print(reversa_cadena("Hola"))

2. Contar la Frecuencia de Caracteres

Este algoritmo cuenta cuántas veces aparece cada carácter en una cadena.

Java


import java.util.HashMap;

public class FrecuenciaCaracteres {
    public static void main(String[] args) {
        String texto = "Hola Mundo";
        HashMap<Character, Integer> frecuencia = new HashMap<>();

        for (char c : texto.toCharArray()) {
            frecuencia.put(c, frecuencia.getOrDefault(c, 0) + 1);
        }

        System.out.println(frecuencia);
    }
}

C++


#include <iostream>
#include <map>
using namespace std;

int main() {
    string texto = "Hola Mundo";
    map<char, int> frecuencia;

    for (char c : texto) {
        frecuencia[c]++;
    }

    for (auto& par : frecuencia) {
        cout << par.first << ": " << par.second << endl;
    }

    return 0;
}

3. Buscar un Substring dentro de otra Cadena

JavaScript


let texto = "Hola Mundo";
let buscar = "Mundo";
console.log(texto.includes(buscar)); // true

Python


texto = "Hola Mundo"
buscar = "Mundo"
print(buscar in texto) # True

4. Eliminar Espacios en Blanco

C#


using System;

class Program {
    static void Main() {
        string texto = " Hola Mundo ";
        Console.WriteLine(texto.Trim());
    }
}

Java


public class Main {
    public static void main(String[] args) {
        String texto = " Hola Mundo ";
        System.out.println(texto.trim());
    }
}
