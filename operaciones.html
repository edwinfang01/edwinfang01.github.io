Principales Operaciones con Cadenas

Las cadenas de texto son una de las estructuras de datos más usadas en programación. Aquí te mostramos las operaciones más comunes con ejemplos en distintos lenguajes.

1. Concatenación de Cadenas

Unir dos cadenas en una sola.

JavaScript


let cadena1 = "Hola";
let cadena2 = "Mundo";
console.log(cadena1 + " " + cadena2); // "Hola Mundo"

Python


cadena1 = "Hola"
cadena2 = "Mundo"
print(cadena1 + " " + cadena2) # "Hola Mundo"

2. Obtener la Longitud de una Cadena

Determina cuántos caracteres tiene una cadena.

JavaScript


let texto = "Hola Mundo";
console.log(texto.length); // 10

Python


texto = "Hola Mundo"
print(len(texto)) # 10

3. Convertir a Mayúsculas y Minúsculas

Convierte el texto a mayúsculas o minúsculas.

Java


String texto = "Hola Mundo";
System.out.println(texto.toUpperCase()); // "HOLA MUNDO"
System.out.println(texto.toLowerCase()); // "hola mundo"

C#


string texto = "Hola Mundo";
Console.WriteLine(texto.ToUpper()); // "HOLA MUNDO"
Console.WriteLine(texto.ToLower()); // "hola mundo"

4. Extraer Subcadenas

Obtener parte de una cadena.

JavaScript


let texto = "Hola Mundo";
console.log(texto.substring(0, 4)); // "Hola"

Python


texto = "Hola Mundo"
print(texto[0:4]) # "Hola"

5. Reemplazar Texto en una Cadena

Reemplaza una parte de la cadena por otra.

JavaScript


let texto = "Hola Mundo";
console.log(texto.replace("Mundo", "JavaScript")); // "Hola JavaScript"

Python


texto = "Hola Mundo"
print(texto.replace("Mundo", "Python")) # "Hola Python"

6. Eliminar Espacios en Blanco

Quita los espacios al inicio y al final de una cadena.

Java


String texto = "  Hola Mundo  ";
System.out.println(texto.trim()); // "Hola Mundo"

C++


#include <iostream>
#include <algorithm>
using namespace std;

string trim(const string &texto) {
    size_t inicio = texto.find_first_not_of(" ");
    size_t fin = texto.find_last_not_of(" ");
    return texto.substr(inicio, fin - inicio + 1);
}

int main() {
    string texto = "  Hola Mundo  ";
    cout << trim(texto) << endl; // "Hola Mundo"
    return 0;
}
